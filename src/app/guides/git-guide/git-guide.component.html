Short git diff & git checkout Guide

Introduction

Introducing a local git repository with 5 commits. Use of git log to check the commits history and get the hash number of each commit.

Getting the differences between commits

Use of git diff to check the difference between commits.

Syntax as follows: git diff <hash of the first commit to compare><hash of the second commit to compare

Example: git diff 23eb1 dc720

Switching between commits

Syntax as follows: git checkout <hash of the destination commit>

Example: git diff dc720
